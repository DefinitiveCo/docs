## Key features

- **Instant quotes** — real-time pricing, fee breakdown, and routing for any supported token pair.
- **Wallet signing** — quotes return a typed-data payload; the swapper signs with their wallet, and Definitive executes.
- **MEV protection** — orders are settled through private channels to avoid front-running.

## Flow

```mermaid
sequenceDiagram
    participant Partner
    participant Flash API
    participant Blockchain

    Partner->>Flash API: POST /quote
    Flash API-->>Partner: Quote (quoteId, pricing, typed-data)
    Partner->>Partner: Sign typed-data with wallet
    Partner->>Flash API: POST /order (quoteId + signature)
    Flash API-->>Partner: Order confirmation
    Flash API->>Blockchain: Execute swap (MEV-protected)
```
