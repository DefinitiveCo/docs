## System status

[https://status.definitive.fi/](https://status.definitive.fi/) reports real-time Flash API uptime. Check it out if requests are failing to rule out a platform-wide issue.

## Flash Dashboard

[https://app.definitive.fi/flash-dashboard](https://app.definitive.fi/flash-dashboard) tracks your integration's trading activity in near real time. It surfaces:

- **Headline metrics**: volume (versus the prior period), fees accrued in your fee vault, order count with average notional
- **Volume by chain**: traded volume broken down per chain
- **Order history**: search and filter your orders
- **Key management**: create and delete your API keys
